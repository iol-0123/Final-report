我的心得

在這次基礎網頁設計的課程中，我學會了如何使用 HTML 建立網頁的基本架構、利用 CSS 美化版面，甚至用 JavaScript 增加互動效果。起初我對寫程式感到有點陌生，但隨著一次次練習，從簡單的標籤、排版，到能讓按鈕改變畫面內容，我慢慢體會到網頁設計的樂趣與成就感。
這份期末報告讓我重新整理了整學期所學的內容，也挑戰自己將圖片、按鈕與版面設計結合在一起。雖然在排版與格式上有時會遇到錯誤，但每次修改後看到成果，我都覺得很有收穫。
